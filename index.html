本地部署方法

  1. 克隆此仓库

     git clone https://github.com/mafengshe-book/web-senior.git

  2. 安装依赖

     npm install

  3. 运行

     npm start

  4. 构建

     npm run build

  5. 部署

     npm run deploy

部署前请务必先构建
